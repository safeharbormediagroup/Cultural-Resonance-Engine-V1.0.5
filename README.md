## 🚦 GETTING STARTED

### 1. **Clone the Repository**
```bash
git clone https://github.com/safeharbormediagroup/Cultural-Resonance-Engine-V1.0.5.git
cd Cultural-Resonance-Engine-V1.0.5
```

### 2. **Compile the Engine**
```bash
g++ main.cpp -o energetic_resonance_compiler
```

### 3. **Run the Resonance Compiler**
```bash
./energetic_resonance_compiler
```

### 4. **Expected Output**
The engine will parse a simulated trading intent log stream and output:
- Detected vibe profiles
- Matched music genres & slang signatures
- Pop art symbolism assignments
- Frequency harmonization values
- Final JSON ledger payload ready for tokenization
